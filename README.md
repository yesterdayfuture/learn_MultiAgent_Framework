# learn_MultiAgent_Framework

学习大模型中的多智能体协作框架，本项目包含多个子项目，分别学习和实践不同的多智能体框架和工具。

## 项目列表

| 项目名称 | 技术栈 | 功能描述 |
|---------|--------|----------|
| autoGenTest | Python, AutoGen | 学习和实践 AutoGen 框架的各种功能，包括模型调用、多模态消息、AssistantAgent、函数转换工具、MCP、上下文管理、多模型代理、团队协作、Swarm 团队、MagenticOne 团队、记忆系统等 |
| testLangfuse | Python, Langfuse | 学习和实践 Langfuse 的使用，用于监控和追踪大模型请求调用，包含部署安装指南和 OpenAI 集成示例 |
| use_langgraph | Python, LangGraph | 学习和实践 LangGraph 框架，实现基础多智能体系统，支持流式和非流式输出、回调机制和装饰器模式 |

## 项目详细说明

### 1. autoGenTest

**技术栈：** Python, AutoGen

**功能：**
- 模型调用与多模态消息处理
- AssistantAgent 的使用
- 自动函数转换工具
- MCP (Model Context Protocol) 集成
- 上下文管理和状态维护
- 多模型代理配置
- 团队协作机制
- Swarm 团队模式
- MagenticOne 团队编排
- 记忆系统实现

### 2. testLangfuse

**技术栈：** Python, Langfuse

**功能：**
- Langfuse 部署和安装指南
- 大模型请求监控和追踪
- OpenAI API 集成与会话管理
- 性能指标收集和分析

### 3. use_langgraph

**技术栈：** Python, LangGraph, LangChain

**功能：**
- 基础多智能体系统构建
- 状态图 (StateGraph) 设计
- 流式和非流式输出支持
- 回调机制实现
- 装饰器模式应用
- 多智能体协作流程编排

## 快速开始

每个子项目都包含独立的示例代码和说明文档，请进入对应目录查看详细的使用方法。
